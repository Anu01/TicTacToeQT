# Tic Tac Toe using QT & c++.
It is an easy game made using QT & C++ , QT using Qmake.
INSTALLATION-
1)Clone the repository
2)Open the .pro file from QTCreator.
3)Build the program.
4)Run the executed file

Installation Using terminal
1)Clone the repository
2)cd TicTacToeQtT/
3)mkdir build/
4)cd build/
5)qmake ../TicTacToe.pro
6)make
7)./TicTacToe
